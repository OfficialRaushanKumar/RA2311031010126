# Screenshots

Place your submission screenshots in this folder and commit them to the repository.

Recommended filenames:
- 1_register_filled.png
- 2_register_success.png
- 3_login_filled.png
- 4_dashboard.png
- 5_logout.png
- 6_postman_register.png
- 7_postman_login.png
- 8_postman_profile.png
- 9_github_repo.png

How to add and push:
1. Copy or move the image files into this folder.
2. Run:
   git add screenshots/*
   git commit -m "Add submission screenshots"
   git push

Notes:
- Keep images reasonably sized (under ~5MB each) to avoid large pushes.
- If you don't yet have screenshots, keep the `.gitkeep` file so Git tracks the folder.
